# OpenGL (stage-2 wrapper)

The scripts under `stage-2/system/opengl/` are **compatibility wrappers** that
forward to the canonical stage-1 installers and assets.

- Canonical docs: `stage-1/system/opengl/README.md`
- Canonical scripts: `stage-1/system/opengl/setup-opengl-win32.sh`
- Canonical assets:
  - `stage-1/system/opengl/10_nvidia.json`
  - `stage-1/system/opengl/docker-compose-win32.yml`

Prefer referencing `stage-1/system/opengl/...` in `user_config.yml`.
