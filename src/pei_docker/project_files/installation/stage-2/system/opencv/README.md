# OpenCV (stage-2 wrapper)

The scripts under `stage-2/system/opencv/` are **compatibility wrappers** that
forward to the canonical stage-1 installers.

- Canonical docs: `stage-1/system/opencv/README.md`
- Canonical scripts: `stage-1/system/opencv/*.sh`

Prefer referencing `stage-1/system/opencv/...` in `user_config.yml`.
