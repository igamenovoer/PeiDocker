# Bun (stage-2 wrapper)

The scripts under `stage-2/system/bun/` are **compatibility wrappers** that forward
to the canonical stage-1 installers.

- Canonical docs: `stage-1/system/bun/README.md`
- Canonical scripts: `stage-1/system/bun/install-bun.sh`

Prefer referencing `stage-1/system/bun/...` in `user_config.yml`.
