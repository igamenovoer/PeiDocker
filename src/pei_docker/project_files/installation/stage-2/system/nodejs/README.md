# Node.js (stage-2 wrapper)

The scripts under `stage-2/system/nodejs/` are **compatibility wrappers** that
forward to the canonical stage-1 installers.

- Canonical docs: `stage-1/system/nodejs/README.md`
- Canonical scripts: `stage-1/system/nodejs/*.sh`

Prefer referencing `stage-1/system/nodejs/...` in `user_config.yml`.
