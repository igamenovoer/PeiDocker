# LiteLLM (stage-2 wrapper)

The scripts under `stage-2/system/litellm/` are **compatibility wrappers** that
forward to the canonical stage-1 installers.

- Canonical docs: `stage-1/system/litellm/README.md`
- Canonical scripts: `stage-1/system/litellm/install-litellm.sh`
- Canonical assets: `stage-1/system/litellm/proxy.py`

Prefer referencing `stage-1/system/litellm/...` in `user_config.yml`.
