# Pixi (Stage-2 Wrappers)

The scripts under `stage-2/system/pixi/` are **compatibility wrappers** that
forward to the stage-1 canonical scripts.

- Canonical docs: `stage-1/system/pixi/README.md`
- Canonical scripts: `stage-1/system/pixi/*.bash`

Prefer referencing `stage-1/system/pixi/...` in `user_config.yml`.
