# Claude Code (stage-2 wrapper)

The scripts under `stage-2/system/claude-code/` are **compatibility wrappers** that
forward to the canonical stage-1 installers.

- Canonical docs: `stage-1/system/claude-code/README.md`
- Canonical scripts: `stage-1/system/claude-code/install-claude-code.sh`

Prefer referencing `stage-1/system/claude-code/...` in `user_config.yml`.
