# Codex CLI (stage-2 wrapper)

The scripts under `stage-2/system/codex-cli/` are **compatibility wrappers** that
forward to the canonical stage-1 installers.

- Canonical docs: `stage-1/system/codex-cli/README.md`
- Canonical scripts: `stage-1/system/codex-cli/install-codex-cli.sh`

Prefer referencing `stage-1/system/codex-cli/...` in `user_config.yml`.
